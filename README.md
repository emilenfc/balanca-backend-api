BALANCA API is a backend service built with **Go**, **Gin**, **GORM**, and **PostgreSQL** that provides a transparent, auditable, and scalable system for managing **personal and shared finances**.

The API enables users to track income, expenses, balances, and financial responsibilities across **personal accounts** and **multiple groups** (such as home, family, or projects). It supports **role-based access control**, **group-based budgeting**, **immutable financial transactions**, and **audit logging** to ensure accountability and clarity.

---

## Core Features

- **User Authentication & Authorization**
  - Secure authentication and role-based access control

- **Personal Finance Management**
  - Personal (self) income tracking
  - Personal expense tracking
  - Automatic balance updates

- **Group Finance Management**
  - Group creation and management
  - Member invitations and role assignment
  - Shared group balance tracking

- **Contributions Management**
  - Member-based contributions
  - External source contributions

- **Expense Tracking**
  - Planned expenses
  - Paid expense recording
  - Payment attribution (who paid and when)

- **Audit & Transparency**
  - Immutable financial transactions
  - Full transaction history
  - Audit logs for sensitive actions

- **Reporting Support**
  - Reporting-ready data structures
  - Monthly and period-based financial analysis

---

## System Philosophy

BALANCA is designed to eliminate ambiguity around money usage by ensuring that **every transaction has a clear source, purpose, owner, and timestamp**.

This approach replaces assumptions and disputes with **data-driven financial clarity**.
